Lez4_3 — Cicli annidati con la Tabella pitagorica

Obiettivo

Comprendere l’annidamento di cicli: un ciclo esterno sulle righe (i) e un ciclo interno sulle colonne (j) per generare la tabella pitagorica 1..N.

  • Diagrammi di flusso + pseudocodice per while, do–while, for
  • Tabella di traccia (N=3) per seguire l’evoluzione di i, j e i·j

Schema mentale: per ogni riga i, esegui tutte le colonne j. Finito l’anello interno → newline → incrementa i.

1) While annidati — tabelline 1..N

Flowchart — while esterno + while interno

Pseudocodice (while/while)

Input N
i ← 1
while i ≤ N do
    j ← 1
    while j ≤ N do
        stampa(i·j)      // prodotto in riga
        j ← j + 1
    fine while
    stampa(newline)      // fine riga
    i ← i + 1
fine while

2) Do–while annidati — tabelline 1..N

Pseudocodice (do–while/do–while)

Input N
i ← 1
ripeti
    j ← 1
    ripeti
        stampa(i·j)
        j ← j + 1
    finché j ≤ N
    stampa(newline)
    i ← i + 1
finché i ≤ N

do–while / repeat–until esegue il corpo almeno una volta; la condizione è verificata alla fine.

Pseudocodice (for/for)

Input N
per i da 1 a N fai
    per j da 1 a N fai
        stampa(i·j)
    fine per
    stampa(newline)
fine per

Invariante (intuizione): all’inizio dell’iterazione con valore j, in quella riga sono stati stampati i prodotti i·1, i·2, …, i·(j−1).

Traccia dell’esecuzione (esempio N = 3)

Per fissare le idee: ad ogni i fisso, il ciclo interno j scorre 1..N e stampa i·j. Alla fine della riga, newline e incremento di i.

Varianti didattiche (facoltative)

  • Triangolare superiore: nel ciclo interno termina a j ≤ i (stampi metà tabella).
  • Solo pari: stampa solo se (i·j) mod 2 = 0.
  • Formattazione: colonne a larghezza fissa per allineare visivamente i prodotti.
