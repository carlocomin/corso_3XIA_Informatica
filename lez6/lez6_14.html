Esercizi avanzati — Piccoli programmi completi

Obiettivo della lezione

In questa lezione affrontiamo esercizi che non sono più “una sola cosa”, ma un mini-programma con più funzionalità combinate:

  • uso combinato di if, while, for, switch;
  • gestione di più variabili di stato (es. saldo di un conto);
  • array come contenitore di dati realistici (voti di una classe);
  • controllo dei tentativi e validazione input.

Qui siamo già nel territorio “scrivo un programma che fai girare davvero”. Questo è l’obiettivo pratico del corso.

E1 — Verifica password con tentativi limitati

Testo: Salva una password corretta (es. "abc123"). Chiedi all’utente di inserirla. L’utente ha massimo 3 tentativi. Se indovina prima dei 3 tentativi → "Accesso consentito". Se dopo 3 tentativi ancora sbaglia → "Bloccato".

Competenze: ciclo con contatore tentativi, confronto stringhe, if annidati.

// C++
// NOTA: qui usiamo std::string per semplicità.
// In C++ l'operatore == su std::string confronta il contenuto testuale.
#include <iostream>
#include <string>
int main() {
    const std::string password_corretta = "abc123";
    std::string tentativo;

    bool accesso = false;

    for (int tent = 1; tent <= 3; tent++) {
        std::cout << "Inserisci password (" << tent
                  << "/3): ";
        std::cin >> tentativo;

        if (tentativo == password_corretta) {
            accesso = true;
            break;
        } else {
            std::cout << "Errata.\n";
        }
    }

    if (accesso) {
        std::cout << "Accesso consentito.\n";
    } else {
        std::cout << "Bloccato.\n";
    }
    return 0;
}
// C
// In C le stringhe sono array di char terminati da '\0'.
// Per confrontarle si usa strcmp, non l'operatore ==.
#include <stdio.h>
#include <string.h>

int main(void) {
    const char password_corretta[] = "abc123";
    char tentativo[100];
    int accesso = 0;

    for (int tent = 1; tent <= 3; tent++) {
        printf("Inserisci password (%d/3): ", tent);
        scanf("%99s", tentativo);

        if (strcmp(tentativo, password_corretta) == 0) {
            accesso = 1;
            break;
        } else {
            printf("Errata.\n");
        }
    }

    if (accesso) {
        printf("Accesso consentito.\n");
    } else {
        printf("Bloccato.\n");
    }
    return 0;
}

Questo schema (tentativi limitati) è tipico della sicurezza base: non lasci tentativi infiniti.

E2 — Statistiche su una classe

Testo: Leggi i voti di 10 studenti in un array di interi. Calcola e stampa:

  • media;
  • voto massimo e minimo;
  • quanti voti < 6 (insufficienti).

Competenze: scansione array multipla, accumulo di statistiche, analisi di dati.

// C++
#include <iostream>
int main() {
    const int N = 10;
    int voti[N];

    for (int i = 0; i < N; i++) {
        std::cout << "Voto studente " << i << ": ";
        std::cin >> voti[i];
    }

    int somma = 0;
    int min = voti[0];
    int max = voti[0];
    int insuff = 0;

    for (int i = 0; i < N; i++) {
        somma += voti[i];
        if (voti[i] < min) min = voti[i];
        if (voti[i] > max) max = voti[i];
        if (voti[i] < 6)   insuff++;
    }

    double media = (double)somma / N;

    std::cout << "Media classe = " << media << "\n";
    std::cout << "Min = " << min << ", Max = " << max << "\n";
    std::cout << "Insufficienti (<6): " << insuff << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 10;
    int voti[N];
    int i;

    for (i = 0; i < N; i++) {
        printf("Voto studente %d: ", i);
        scanf("%d", &voti[i]);
    }

    int somma = 0;
    int min = voti[0];
    int max = voti[0];
    int insuff = 0;

    for (i = 0; i < N; i++) {
        somma += voti[i];
        if (voti[i] < min) min = voti[i];
        if (voti[i] > max) max = voti[i];
        if (voti[i] < 6)   insuff++;
    }

    double media = (double)somma / N;

    printf("Media classe = %.2f\n", media);
    printf("Min = %d, Max = %d\n", min, max);
    printf("Insufficienti (<6): %d\n", insuff);
    return 0;
}

Questo è un “mini report”: prende dati grezzi e produce statistiche utili. È esattamente ciò che fanno tanti programmi di analisi.

E3 — Normalizzazione dei valori

Testo: Leggi 5 numeri interi in un array. Trova il valore massimo. Per ogni valore stampa il rapporto valore / massimo come numero in virgola mobile (double, quindi con la parte decimale).

Competenze: trovare max, attenzione alla divisione intera vs divisione reale, cast a double.

// C++
#include <iostream>
int main() {
    const int N = 5;
    int v[N];

    for (int i = 0; i < N; i++) {
        std::cout << "v[" << i << "] = ";
        std::cin >> v[i];
    }

    int massimo = v[0];
    for (int i = 1; i < N; i++) {
        if (v[i] > massimo) massimo = v[i];
    }

    std::cout << "Valori normalizzati (val/max):\n";
    for (int i = 0; i < N; i++) {
        double ratio = (double)v[i] / massimo;
        std::cout << v[i] << " -> " << ratio << "\n";
    }

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 5;
    int v[N];
    int i;

    for (i = 0; i < N; i++) {
        printf("v[%d] = ", i);
        scanf("%d", &v[i]);
    }

    int massimo = v[0];
    for (i = 1; i < N; i++) {
        if (v[i] > massimo) massimo = v[i];
    }

    printf("Valori normalizzati (val/max):\n");
    for (i = 0; i < N; i++) {
        double ratio = (double)v[i] / massimo;
        printf("%d -> %f\n", v[i], ratio);
    }

    return 0;
}

Questo è fondamentale in analisi dati: riportare i valori tutti nella scala [0,1].

E4 — Frequenza delle cifre (0..9)

Testo: Chiedi un numero intero “grande” all’utente, trattandolo come testo (stringa). Conta quante volte compare ciascuna cifra da 0 a 9. Stampa il risultato.

Competenze: array come contatore di frequenze, scansione di una stringa char per char, indicizzazione tramite valore numerico della cifra.

// C++
// Strategia:
// 1. leggo un numero come stringa (es "5012227")
// 2. creo un array freq[10] inizializzato a 0
// 3. per ogni carattere c, calcolo cifra = c - '0'
// 4. incremento freq[cifra]
#include <iostream>
#include <string>
int main() {
    std::string numero;
    std::cout << "Inserisci un numero intero grande: ";
    std::cin >> numero;

    int freq[10] = {0}; // tutte le frequenze inizialmente 0

    for (char c : numero) {
        if (c >= '0' && c <= '9') {
            int cifra = c - '0'; // '0'->0, '1'->1, ..., '9'->9
            freq[cifra]++;
        }
    }

    std::cout << "Frequenze cifre:\n";
    for (int d = 0; d < 10; d++) {
        std::cout << d << ": " << freq[d] << "\n";
    }
    return 0;
}
// C
// In C non abbiamo std::string, ma possiamo leggere una stringa in un array di char.
// Poi facciamo la stessa scansione e usiamo un array freq[10] di contatori.
#include <stdio.h>

int main(void) {
    char numero[256];
    int freq[10] = {0};
    int i;

    printf("Inserisci un numero intero grande: ");
    scanf("%255s", numero);

    for (i = 0; numero[i] != '\0'; i++) {
        char c = numero[i];
        if (c >= '0' && c <= '9') {
            int cifra = c - '0';
            freq[cifra]++;
        }
    }

    printf("Frequenze cifre:\n");
    for (i = 0; i < 10; i++) {
        printf("%d: %d\n", i, freq[i]);
    }

    return 0;
}

Questo è già “analisi dati”. L’array freq è un istogramma: conta quante volte compare ciascuna categoria (qui: le cifre).

E5 — Somma riga per riga di una matrice 3×3

Testo: Leggi una matrice 3×3 di interi. Per ogni riga calcola la somma degli elementi di quella riga e stampala.

Competenze: cicli annidati su una matrice, uso di una variabile “accumulatore” che si azzera a ogni riga.

// C++
#include <iostream>
int main() {
    int mat[3][3];

    // input matrice
    for (int r = 0; r < 3; r++) {
        for (int c = 0; c < 3; c++) {
            std::cout << "mat[" << r << "][" << c << "] = ";
            std::cin >> mat[r][c];
        }
    }

    // somma per riga
    for (int r = 0; r < 3; r++) {
        int somma_riga = 0;
        for (int c = 0; c < 3; c++) {
            somma_riga += mat[r][c];
        }
        std::cout << "Somma riga " << r
                  << " = " << somma_riga << "\n";
    }

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int mat[3][3];
    int r, c;

    // input matrice
    for (r = 0; r < 3; r++) {
        for (c = 0; c < 3; c++) {
            printf("mat[%d][%d] = ", r, c);
            scanf("%d", &mat[r][c]);
        }
    }

    // somma per riga
    for (r = 0; r < 3; r++) {
        int somma_riga = 0;
        for (c = 0; c < 3; c++) {
            somma_riga += mat[r][c];
        }
        printf("Somma riga %d = %d\n", r, somma_riga);
    }

    return 0;
}

Questo esercizio è molto vicino a operazioni su tabelle (es. fogli di calcolo).

E6 — Valore massimo in una matrice 3×3 e sue coordinate

Testo: Leggi una matrice 3×3 di interi. Trova l’elemento massimo e stampa:

  • il valore massimo;
  • la sua riga e colonna (indice r,c).

Competenze: confronto in doppio ciclo annidato, memorizzazione della posizione “migliore finora”.

// C++
#include <iostream>
int main() {
    int mat[3][3];

    for (int r = 0; r < 3; r++) {
        for (int c = 0; c < 3; c++) {
            std::cout << "mat[" << r << "][" << c << "] = ";
            std::cin >> mat[r][c];
        }
    }

    int maxVal = mat[0][0];
    int maxR = 0;
    int maxC = 0;

    for (int r = 0; r < 3; r++) {
        for (int c = 0; c < 3; c++) {
            if (mat[r][c] > maxVal) {
                maxVal = mat[r][c];
                maxR = r;
                maxC = c;
            }
        }
    }

    std::cout << "Massimo = " << maxVal
              << " in posizione (" << maxR
              << "," << maxC << ")\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int mat[3][3];
    int r, c;

    for (r = 0; r < 3; r++) {
        for (c = 0; c < 3; c++) {
            printf("mat[%d][%d] = ", r, c);
            scanf("%d", &mat[r][c]);
        }
    }

    int maxVal = mat[0][0];
    int maxR = 0;
    int maxC = 0;

    for (r = 0; r < 3; r++) {
        for (c = 0; c < 3; c++) {
            if (mat[r][c] > maxVal) {
                maxVal = mat[r][c];
                maxR = r;
                maxC = c;
            }
        }
    }

    printf("Massimo = %d in posizione (%d,%d)\n", maxVal, maxR, maxC);
    return 0;
}

Questo schema (tieni traccia del "migliore finora" + le sue coordinate) è fondamentale nella ricerca su matrici e immagini.

E7 — Menù robusto con validazione input

Testo: Realizza un menù numerato (1..4). Chiedi la scelta all’utente. Se l’utente inserisce un numero fuori da 1..4, stampa “Scelta non valida” e richiedi di nuovo senza uscire.

Competenze: ciclo do-while o while di validazione, switch, controllo dell’intervallo valido.

// C++
#include <iostream>
int main() {
    int scelta = 0;

    do {
        std::cout << "\nMenu:\n"
                  << "1) Saluta\n"
                  << "2) Mostra data finta\n"
                  << "3) Mostra numero random finto\n"
                  << "4) Esci\n"
                  << "Scelta (1-4): ";
        std::cin >> scelta;

        if (scelta < 1 || scelta > 4) {
            std::cout << "Scelta non valida\n";
            continue;
        }

        switch (scelta) {
        case 1:
            std::cout << "Ciao!\n";
            break;
        case 2:
            std::cout << "Oggi e' 01/01/2099\n";
            break;
        case 3:
            std::cout << "Numero segreto: 12345\n";
            break;
        case 4:
            std::cout << "Chiusura menù...\n";
            break;
        }
    } while (scelta != 4);

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int scelta = 0;

    do {
        printf("\nMenu:\n");
        printf("1) Saluta\n");
        printf("2) Mostra data finta\n");
        printf("3) Mostra numero random finto\n");
        printf("4) Esci\n");
        printf("Scelta (1-4): ");
        scanf("%d", &scelta);

        if (scelta < 1 || scelta > 4) {
            printf("Scelta non valida\n");
            continue;
        }

        switch (scelta) {
        case 1:
            printf("Ciao!\n");
            break;
        case 2:
            printf("Oggi e' 01/01/2099\n");
            break;
        case 3:
            printf("Numero segreto: 12345\n");
            break;
        case 4:
            printf("Chiusura menù...\n");
            break;
        }
    } while (scelta != 4);

    return 0;
}

Questo esercizio è importante perché mostra come “tenere vivo” un programma finché l’utente non chiede di uscire, e come intercettare scelte non valide senza far crashare tutto.

E8 — Triangolo di asterischi

Testo: Chiedi un numero N. Stampa un triangolo di asterischi: 

*
**
***
...
fino a N righe.

Competenze: cicli annidati con rapporto “riga → quante colonne stampo”.

// C++
#include <iostream>
int main() {
    int N;
    std::cout << "N: ";
    std::cin >> N;

    for (int r = 1; r <= N; r++) {
        for (int c = 1; c <= r; c++) {
            std::cout << "*";
        }
        std::cout << "\n";
    }
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int N;
    int r, c;
    printf("N: ");
    scanf("%d", &N);

    for (r = 1; r <= N; r++) {
        for (c = 1; c <= r; c++) {
            printf("*");
        }
        printf("\n");
    }
    return 0;
}

Questo esercizio ti fa “pensare in 2D”: riga e colonna sono due contatori diversi, con limiti diversi (la colonna arriva fino alla riga).

E9 — Parola palindroma (versione base)

Testo: Chiedi una parola senza spazi. Verifica se è palindroma, cioè se è uguale letta da sinistra a destra e da destra a sinistra (es. "anna", "osso"). Stampa "palindroma" o "non palindroma".

Competenze: confronto di caratteri a coppie, indice che parte da inizio e indice che parte da fine.

// C++
// Strategia:
// i parte da 0, j parte da lunghezza-1
// Finché i<j confronta parola[i] e parola[j]
// Se trovi differenza -> non palindroma
#include <iostream>
#include <string>
int main() {
    std::string parola;
    std::cout << "Parola (senza spazi): ";
    std::cin >> parola;

    bool pal = true;
    int i = 0;
    int j = (int)parola.size() - 1;

    while (i < j) {
        if (parola[i] != parola[j]) {
            pal = false;
            break;
        }
        i++;
        j--;
    }

    if (pal) {
        std::cout << "palindroma\n";
    } else {
        std::cout << "non palindroma\n";
    }
    return 0;
}
// C
#include <stdio.h>
#include <string.h>
int main(void) {
    char parola[256];
    printf("Parola (senza spazi): ");
    scanf("%255s", parola);

    int i = 0;
    int j = (int)strlen(parola) - 1;
    int pal = 1; // 1 = palindroma finora

    while (i < j) {
        if (parola[i] != parola[j]) {
            pal = 0;
            break;
        }
        i++;
        j--;
    }

    if (pal) {
        printf("palindroma\n");
    } else {
        printf("non palindroma\n");
    }
    return 0;
}

Qui applichi un ragionamento algoritmico: non serve creare la stringa inversa completa, basta confrontare caratteri “a specchio”.

E10 — Mini-bancomat

Testo: Simula un piccolo bancomat. Parti con un saldo iniziale (es. 1000). Mostra un menù:

  1. Mostra saldo
  2. Prelievo
  3. Versamento
  4. Esci
Regole:
  • il prelievo non può portare il saldo sotto 0 (se succede, rifiuta l’operazione);
  • versamento aggiunge denaro al saldo;
  • il menù si ripete finché l’utente non sceglie “Esci”.

Competenze: stato persistente (saldo), switch in un ciclo, controllo degli errori logici.

// C++
// Nota: qui stiamo simulando un sistema reale.
// Abbiamo uno "stato" (il saldo) che sopravvive tra una scelta e la successiva.
#include <iostream>
int main() {
    double saldo = 1000.0;
    int scelta;

    do {
        std::cout << "\n--- BANCOMAT ---\n"
                  << "1) Mostra saldo\n"
                  << "2) Prelievo\n"
                  << "3) Versamento\n"
                  << "4) Esci\n"
                  << "Scelta: ";
        std::cin >> scelta;

        if (scelta == 1) {
            std::cout << "Saldo attuale: " << saldo << " euro\n";
        } else if (scelta == 2) {
            double quantita;
            std::cout << "Importo da prelevare: ";
            std::cin >> quantita;
            if (quantita <= saldo && quantita > 0) {
                saldo -= quantita;
                std::cout << "Prelievo effettuato. Nuovo saldo: "
                          << saldo << " euro\n";
            } else {
                std::cout << "Operazione rifiutata (fondi insufficienti o importo non valido).\n";
            }
        } else if (scelta == 3) {
            double quantita;
            std::cout << "Importo da versare: ";
            std::cin >> quantita;
            if (quantita > 0) {
                saldo += quantita;
                std::cout << "Versamento effettuato. Nuovo saldo: "
                          << saldo << " euro\n";
            } else {
                std::cout << "Importo non valido.\n";
            }
        } else if (scelta == 4) {
            std::cout << "Uscita...\n";
        } else {
            std::cout << "Scelta non valida.\n";
        }

    } while (scelta != 4);

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    double saldo = 1000.0;
    int scelta;

    do {
        printf("\n--- BANCOMAT ---\n");
        printf("1) Mostra saldo\n");
        printf("2) Prelievo\n");
        printf("3) Versamento\n");
        printf("4) Esci\n");
        printf("Scelta: ");
        scanf("%d", &scelta);

        if (scelta == 1) {
            printf("Saldo attuale: %.2f euro\n", saldo);

        } else if (scelta == 2) {
            double quantita;
            printf("Importo da prelevare: ");
            scanf("%lf", &quantita);
            if (quantita <= saldo && quantita > 0) {
                saldo -= quantita;
                printf("Prelievo effettuato. Nuovo saldo: %.2f euro\n", saldo);
            } else {
                printf("Operazione rifiutata (fondi insufficienti o importo non valido).\n");
            }

        } else if (scelta == 3) {
            double quantita;
            printf("Importo da versare: ");
            scanf("%lf", &quantita);
            if (quantita > 0) {
                saldo += quantita;
                printf("Versamento effettuato. Nuovo saldo: %.2f euro\n", saldo);
            } else {
                printf("Importo non valido.\n");
            }

        } else if (scelta == 4) {
            printf("Uscita...\n");

        } else {
            printf("Scelta non valida.\n");
        }

    } while (scelta != 4);

    return 0;
}

Questo è praticamente un programma gestionale minimale. Mantiene dati tra le scelte dell’utente, controlla la validità delle operazioni, e termina solo quando l’utente sceglie “Esci”.

In sintesi
  • Hai usato array, cicli, condizioni e menù per costruire piccoli “prodotti” funzionanti.
  • Hai gestito input non valido (password sbagliata, scelta sbagliata, prelievo impossibile).
  • Hai ragionato sullo stato: una variabile (saldo, tentativi, ecc.) che vive per tutto il programma.
  • Hai visto matrici e analisi statistica dei dati.
A questo punto sei in grado di leggere un problema descritto a parole e trasformarlo in un algoritmo C/C++ che interagisce con l’utente.
