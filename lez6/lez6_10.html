Il ciclo for in C e C++

Cos’è un ciclo for?

Il ciclo for è una forma compatta di ciclo controllato da contatore. Si usa quando conosciamo in anticipo quante volte vogliamo ripetere un blocco di istruzioni.

La sua sintassi è: 

for (inizializzazione; condizione; aggiornamento) {
    // corpo del ciclo
}
  • inizializzazione: viene eseguita una sola volta, all’inizio;
  • condizione: viene valutata prima di ogni iterazione;
  • aggiornamento: eseguito alla fine di ogni giro.

È l’equivalente compatto di un ciclo while con contatore.

Esempio base in C++: contare da 1 a 5

#include <iostream>
int main() {
    for (int i = 1; i <= 5; i = i + 1) {
        std::cout << i << "\n";
    }
    std::cout << "Fatto!\n";
    return 0;
}

In questo esempio:

  • int i = 1: inizializzazione;
  • i <= 5: condizione di permanenza;
  • i = i + 1: aggiornamento dopo ogni iterazione.

Quando i diventa 6, la condizione è falsa e il ciclo termina.

Esempio equivalente in C

#include <stdio.h>
int main(void) {
    for (int i = 1; i <= 5; i = i + 1) {
        printf("%d\n", i);
    }
    printf("Fatto!\n");
    return 0;
}

Stessa logica: cambia solo la sintassi di stampa (printf invece di std::cout).

Confronto for vs while

forwhile
Tutto il controllo (inizio, condizione, aggiornamento) è concentrato in un’unica riga. Le tre parti sono scritte in punti diversi (prima, dentro e dopo il ciclo).
Più leggibile per cicli con contatore. Più flessibile per condizioni generiche (es. input).

Usa for quando sai già quante volte devi ripetere qualcosa; usa while quando la durata del ciclo dipende da una condizione esterna o da un input.

Uso di break e continue

All’interno di un ciclo possiamo controllare il flusso con due parole chiave:

  • break: interrompe il ciclo immediatamente.
  • continue: salta direttamente all’iterazione successiva.
for (int i = 1; i <= 10; i++) {
    if (i == 5) continue;   // salta il numero 5
    if (i == 9) break;      // ferma il ciclo
    std::cout << i << " ";
}

Output: 1 2 3 4 6 7 8

continue è utile per saltare casi particolari; break serve per interrompere quando raggiungi una condizione di fine anticipata.

Esempio pratico: tabella pitagorica (for annidati)

#include <iostream>
int main() {
    for (int r = 1; r <= 10; r++) {
        for (int c = 1; c <= 10; c++) {
            std::cout << r * c << "\t";
        }
        std::cout << "\n";
    }
    return 0;
}

Questo esempio mostra due cicli annidati:

  • Il ciclo esterno (r) controlla le righe.
  • Il ciclo interno (c) stampa le colonne per ogni riga.

Questo è lo schema tipico per stampare tabelle o scorrere matrici.

Esercizi (con soluzione a comparsa)

  1. E1 — Somma dei numeri da 1 a N
    Testo: Scrivi un programma che chiede N e calcola la somma dei numeri da 1 a N usando for. 
    #include <iostream>
int main() {
    int N, somma = 0;
    std::cout << "Inserisci N: ";
    std::cin >> N;
    for (int i = 1; i <= N; i++) {
        somma += i;
    }
    std::cout << "Somma = " << somma << "\n";
}
  2. E2 — Stampa numeri pari da 2 a 20 
    #include <stdio.h>
int main(void) {
    for (int i = 2; i <= 20; i += 2)
        printf("%d ", i);
    printf("\n");
}
  3. E3 — Conta le cifre di un numero 
    #include <iostream>
int main() {
    int n, cifre = 0;
    std::cout << "Inserisci un numero: ";
    std::cin >> n;
    for (; n > 0; n /= 10) {
        cifre++;
    }
    std::cout << "Numero di cifre: " << cifre << "\n";
}
In sintesi
  • Il ciclo for è perfetto quando conosci il numero di iterazioni.
  • Ha tre parti: inizializzazione, condizione e aggiornamento.
  • È leggibile, compatto e molto usato in programmazione.
  • Può essere annidato per creare strutture a due o più dimensioni.
  • Evita di modificare il contatore dentro il corpo: crea confusione.
