Hello, World — struttura minima di un programma

Obiettivo

  • Comprendere la struttura minima di un programma in C++ e in C.
  • Compilare ed eseguire da riga di comando (GCC/Clang).
  • Capire main, include, stdout, newline \n / std::endl.

Idea chiave: ogni eseguibile inizia da main, scrive su stdout e termina restituendo un int al sistema.

Sezione C++ — il primo programma

#include <iostream>   // dichiara std::cout, std::endl

int main() {
    std::cout << "Hello, World!" << '\n';
    return 0;            // 0 = successo
}
  • #include <iostream>: rende disponibili gli stream standard (std::cout).
  • int main(): punto d’ingresso del programma; il return è il codice di uscita.
  • std::cout: standard output. '\n' manda a capo; std::endl manda a capo e forza il flush (più lento, da usare quando serve).

Compilazione ed esecuzione

g++ -std=c++17 -Wall -Wextra -O0 -g hello.cpp -o hello
./hello    # (Windows: .\hello.exe)

Consiglio: evitando using namespace std; si mantiene il codice più chiaro e sicuro nei nomi (usiamo il prefisso std::).

Sezione C — il primo programma

#include <stdio.h>    // dichiara printf

int main(void) {
    printf("Hello, World!\n");
    return 0;          // 0 = successo
}
  • #include <stdio.h>: funzioni di I/O in stile C (printf/scanf).
  • int main(void): indica che main non riceve argomenti. Anche int main() è accettato.
  • printf: stampa formattata; qui una stringa con newline \n.

Compilazione ed esecuzione

gcc -std=c11 -Wall -Wextra -O0 -g hello.c -o hello
./hello

Stringhe in C: sono array di char terminati da '\0'; printf le interpreta fino a quel terminatore.

Struttura comune (C++ e C)

ElementoSignificatoNote
#include <...> Include dichiarazioni/funzioni di libreria Angolari = libreria standard; virgolette = file locale
int main(...) Punto d’ingresso del programma Restituisce un codice di uscita al SO (0 = ok)
Output Scrive su stdout C++: std::cout; C: printf
Newline Va a capo '\n' in entrambi; std::endl anche flush (C++)

Pitfall comuni (e rimedi pratici)

  • Dimenticare #include → identificatori non trovati.

    Includi <iostream> (C++) o <stdio.h> (C).
  • Output “attaccati” (niente a capo).

    Termina con '\n' (o std::endl se ti serve anche il flush).
  • Usare sempre std::endl (rallenta).

    Preferisci '\n', usa std::endl solo quando ti serve il flush.
  • Encoding/Newline su Windows.

    Sorgenti in UTF-8; per file binari apri in modalità binaria (niente traduzione \n↔\r\n).

Esercizi lampo (con soluzione a comparsa)

  1. E1 — Personalizza il saluto
    Testo: Stampa “Ciao, 3BIA!” in C++ e in C, ciascuno su una riga separata. 
    // C++
#include <iostream>
int main(){
    std::cout << "Ciao, 3BIA!" << '\n';
    return 0;
}
    // C
#include <stdio.h>
int main(void){
    printf("Ciao, 3BIA!\n");
    return 0;
}
  2. E2 — Due righe, due istruzioni
    Testo: In C++ e in C stampa su due righe: “Hello” e poi “World”. 
    // C++
#include <iostream>
int main(){
    std::cout << "Hello" << '\n';
    std::cout << "World" << '\n';
    return 0;
}
    // C
#include <stdio.h>
int main(void){
    printf("Hello\n");
    printf("World\n");
    return 0;
}
  3. E3 — Codice di uscita
    Testo: Fai terminare il programma restituendo il valore 7. Come verifichi da shell che il codice di uscita è 7? 
    // C++ (uguale in C, cambia solo include e stampa)
#include <iostream>
int main(){ return 7; }
    # Linux/macOS:
./hello ; echo $?
# Windows PowerShell:
.\hello.exe ; echo $LASTEXITCODE
Per saperne di più (accenni): esistono varianti di main con argomenti (int main(int argc, char* argv[])) per leggere parametri da riga di comando. Le useremo più avanti.
