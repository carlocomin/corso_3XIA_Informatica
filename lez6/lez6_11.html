Array (vettori) in C e C++

Perché servono gli array?

Finora abbiamo usato variabili singole: 

int voto1;
int voto2;
int voto3;
...

Problema: se devo gestire 30 voti, dovrei fare 30 variabili diverse. È ingestibile.

Un array è una collezione ordinata di elementi tutti dello stesso tipo, memorizzati uno dopo l’altro in memoria, e accessibili tramite un indice.

Quindi invece di 30 variabili voto1, voto2, …, abbiamo una sola variabile: 

int voti[30];   // 30 interi

Ora posso dire “il voto numero 0”, “il voto numero 1”, ecc.

Gli array sono fondamentali per rappresentare liste, tabelle, righe di dati, serie di campioni, ecc.

Dichiarare un array

In C e in C++ (forma base)

int a[5];        // array di 5 interi
double misure[10]; // array di 10 double
char nome[100];    // array di 100 char (per stringhe in stile C)
  • int a[5]; crea 5 celle consecutive di tipo int.
  • Gli indici partono da 0, quindi in a[5] gli elementi validi sono: a[0], a[1], a[2], a[3], a[4].
  • Attenzione: a[5] NON esiste. Andare fuori dai limiti è errore grave (comportamento indefinito).

Suggerimento mentale: se la dimensione è N, l’ultimo indice valido è N-1.

Inizializzazione al momento della dichiarazione

int numeri[4] = {10, 20, 30, 40};

int zero[5] = {0};    // tutti 0: {0,0,0,0,0}

char vocali[5] = {'a','e','i','o','u'};

Possiamo anche lasciare che il compilatore calcoli la dimensione: 

int valori[] = {3, 5, 7, 11}; // dimensione = 4

Questo è molto comodo: se aggiungo/tolgo elementi dalla lista di inizializzazione, la dimensione si aggiorna da sola.

Accedere agli elementi

Per leggere o scrivere in una certa posizione dell’array uso l’indice tra parentesi quadre [ ].

Esempio in C++

#include <iostream>
int main() {
    int a[5];
    a[0] = 10;        // scrittura
    a[1] = 20;

    std::cout << a[0] << "\n"; // lettura (stampa 10)
    std::cout << a[1] << "\n"; // stampa 20

    return 0;
}

Esempio in C

#include <stdio.h>
int main(void) {
    int a[5];
    a[0] = 10;
    a[1] = 20;

    printf("%d\n", a[0]); // stampa 10
    printf("%d\n", a[1]); // stampa 20

    return 0;
}

Se usi un indice che non esiste (per esempio a[1000] in un array da 5 elementi), vai a leggere/scrivere memoria che non è tua → comportamento imprevedibile, potenzialmente crash.

Riempire e stampare un array con un ciclo for

In pratica, quasi sempre useremo un ciclo per “visitare” tutti gli elementi dell’array. Questo è un pattern fondamentale che useremo per tutta l’informatica.

C++ — leggi 5 numeri dall’utente e poi ristampali

#include <iostream>
int main() {
    int v[5];

    // input
    for (int i = 0; i < 5; i++) {
        std::cout << "Inserisci v[" << i << "]: ";
        std::cin >> v[i];
    }

    // output
    std::cout << "Hai inserito: ";
    for (int i = 0; i < 5; i++) {
        std::cout << v[i] << " ";
    }
    std::cout << "\n";

    return 0;
}

C — stesso esercizio

#include <stdio.h>
int main(void) {
    int v[5];
    int i;

    // input
    for (i = 0; i < 5; i++) {
        printf("Inserisci v[%d]: ", i);
        scanf("%d", &v[i]);
    }

    // output
    printf("Hai inserito: ");
    for (i = 0; i < 5; i++) {
        printf("%d ", v[i]);
    }
    printf("\n");

    return 0;
}

Nota lo schema: for (i = 0; i < 5; i++) significa “visita tutte le posizioni valide dell’array v”.

Se un array ha dimensione N, il ciclo tipico è: for (i = 0; i < N; i++). Lo useremo sempre.

Elaborare un array: somma, massimo, media

Una volta che i dati sono nel vettore, possiamo fare calcoli su tutti gli elementi.

Somma e media (C++)

#include <iostream>
int main() {
    const int N = 5;
    int v[N];
    int somma = 0;

    for (int i = 0; i < N; i++) {
        std::cout << "v[" << i << "] = ";
        std::cin >> v[i];
        somma += v[i];
    }

    double media = (double)somma / N;
    std::cout << "Somma = " << somma << "\n";
    std::cout << "Media = " << media << "\n";
    return 0;
}

Ragionamento:

  • Leggiamo tutti i valori in v[i].
  • Aggiungiamo ogni valore a somma.
  • La media è somma diviso quantità di elementi.

Massimo (C)

#include <stdio.h>
int main(void) {
    const int N = 5;
    int v[N];
    int i;

    // input
    for (i = 0; i < N; i++) {
        printf("v[%d] = ", i);
        scanf("%d", &v[i]);
    }

    // ipotizzo che il massimo iniziale sia il primo elemento
    int massimo = v[0];

    for (i = 1; i < N; i++) {
        if (v[i] > massimo) {
            massimo = v[i];
        }
    }

    printf("Il massimo e': %d\n", massimo);
    return 0;
}

Schema generale per trovare il massimo:

  1. salva il primo elemento come “candidato massimo”;
  2. confronta tutti gli altri con il massimo attuale;
  3. se trovi un valore più grande, aggiornalo.

Piccole differenze tra C e C++ (per ora)

Per ora, stiamo usando array “in stile C”: int v[10]; in entrambi i linguaggi.

In C++ moderno esiste anche std::array (serve #include <array>): 

#include <array>
#include <iostream>
int main() {
    std::array<int, 5> v = {1,2,3,4,5};
    for (int i = 0; i < 5; i++) {
        std::cout << v[i] << " ";
    }
    std::cout << "\n";
    return 0;
}

Perché usarlo?

  • std::array “sa” la propria dimensione (puoi fare v.size());
  • funziona bene con funzioni e librerie C++ moderne;
  • è più “sicuro” e leggibile nel codice grande.

Ma a livello basso, std::array<T,N> è comunque un blocco di N elementi contigui, come T v[N];. Per noi, in questa fase, possiamo tranquillamente restare sugli array classici.

Esercizi (con soluzione a comparsa)

  1. E1 — Media della classe
    Testo: Chiedi all’utente di inserire i voti di 5 studenti (numeri interi). Stampa la media dei voti. 
    #include <iostream>
int main() {
    const int N = 5;
    int voti[N];
    int somma = 0;

    for (int i = 0; i < N; i++) {
        std::cout << "Voto " << i << ": ";
        std::cin >> voti[i];
        somma += voti[i];
    }

    double media = (double)somma / N;
    std::cout << "Media classe = " << media << "\n";
    return 0;
}

    Questo esercizio ti fa lavorare su: input in array, somma, media.

  2. E2 — Minimo e massimo
    Testo: Leggi 5 numeri interi in un array (in C). Stampa il valore minimo e massimo. 
    #include <stdio.h>
int main(void) {
    const int N = 5;
    int v[N];
    int i;

    for (i = 0; i < N; i++) {
        printf("v[%d] = ", i);
        scanf("%d", &v[i]);
    }

    int min = v[0];
    int max = v[0];

    for (i = 1; i < N; i++) {
        if (v[i] < min) min = v[i];
        if (v[i] > max) max = v[i];
    }

    printf("Min = %d, Max = %d\n", min, max);
    return 0;
}

    Stiamo facendo due confronti in un unico ciclo: è un’ottima abitudine (efficiente e chiaro).

  3. E3 — Stampa inversa
    Testo: Leggi 5 numeri interi in un array. Poi stampali al contrario (dall’ultimo al primo). 
    #include <iostream>
int main() {
    const int N = 5;
    int v[N];

    for (int i = 0; i < N; i++) {
        std::cout << "v[" << i << "] = ";
        std::cin >> v[i];
    }

    std::cout << "Inverso: ";
    for (int i = N-1; i >= 0; i--) {
        std::cout << v[i] << " ";
    }
    std::cout << "\n";

    return 0;
}

    Attenzione al ciclo in discesa: for (int i = N-1; i >= 0; i--). Se i fosse senza segno (es. unsigned), rischieresti un ciclo infinito. Per ora usa int.

  4. E4 — Trova un valore nell’array
    Testo: Leggi 5 numeri. Poi chiedi un numero X e stampa se X è presente nell’array oppure no. 
    #include <stdio.h>
int main(void) {
    const int N = 5;
    int v[N];
    int i, x;
    int trovato = 0; // 0 = no, 1 = si

    for (i = 0; i < N; i++) {
        printf("v[%d] = ", i);
        scanf("%d", &v[i]);
    }

    printf("Cerca il valore: ");
    scanf("%d", &x);

    for (i = 0; i < N; i++) {
        if (v[i] == x) {
            trovato = 1;
            break; // posso uscire prima
        }
    }

    if (trovato) {
        printf("Presente nell'array.\n");
    } else {
        printf("Non presente.\n");
    }

    return 0;
}

    Questo è chiamato ricerca lineare: controlliamo ogni elemento, uno alla volta, finché troviamo quello che ci interessa.

In sintesi
  • Un array è un blocco di celle tutte dello stesso tipo, indicizzate da 0 a N-1.
  • Si dichiara con tipo nome[dim];.
  • Per accedere uso nome[indice].
  • Scorro un array quasi sempre con un ciclo for che va da 0 a N-1.
  • Posso calcolare somma, media, massimo, minimo, cercare un elemento… tutto con un ciclo.
