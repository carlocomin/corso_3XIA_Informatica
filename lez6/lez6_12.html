Esercizi base — Fondamentali di C e C++

Obiettivo della lezione

Questa lezione raccoglie esercizi “di base” sugli argomenti visti finora:

  • input / output;
  • variabili e tipi base;
  • operatori aritmetici;
  • if / else di base;
  • cicli for e while semplici;
  • array monodimensionali (lettura e stampa).

Ogni esercizio ha la soluzione a comparsa. Prima, però, prova mentalmente (o su carta!) a prevedere cosa faresti tu.

E1 — Saluto personalizzato

Testo: Scrivi un programma che chieda all’utente il suo nome e poi stampi a schermo:
Ciao, <nome>!

Competenze: input, output, stringhe semplici.

// Versione C++
#include <iostream>
#include <string>
int main() {
    std::string nome;
    std::cout << "Come ti chiami? ";
    std::cin >> nome; // legge fino al primo spazio
    std::cout << "Ciao, " << nome << "!\n";
    return 0;
}
// Versione C
#include <stdio.h>
int main(void) {
    char nome[100];
    printf("Come ti chiami? ");
    scanf("%99s", nome); // legge una parola senza spazi, massimo 99 char
    printf("Ciao, %s!\n", nome);
    return 0;
}

Nota: in C usiamo char nome[100] perché le stringhe sono array di char.

E2 — Somma di due numeri interi

Testo: Chiedi all’utente due numeri interi e stampa la loro somma.

Competenze: tipi interi, somma, uso di due variabili, output formattato.

// C++
#include <iostream>
int main() {
    int a, b;
    std::cout << "Inserisci due interi: ";
    std::cin >> a >> b;
    std::cout << "Somma = " << (a + b) << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int a, b;
    printf("Inserisci due interi: ");
    scanf("%d %d", &a, &b);
    printf("Somma = %d\n", a + b);
    return 0;
}

Qui usiamo solo operatori aritmetici base (+) e I/O.

E3 — Area del rettangolo

Testo: Leggi base e altezza (interi positivi), calcola l’area = base × altezza, stampa il risultato.

Competenze: input, prodotto, variabili significative.

// C++
#include <iostream>
int main() {
    int base, altezza;
    std::cout << "Base? ";
    std::cin >> base;
    std::cout << "Altezza? ";
    std::cin >> altezza;

    int area = base * altezza;
    std::cout << "Area = " << area << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int base, altezza;
    printf("Base? ");
    scanf("%d", &base);
    printf("Altezza? ");
    scanf("%d", &altezza);

    int area = base * altezza;
    printf("Area = %d\n", area);
    return 0;
}

Consiglio: scegli sempre nomi di variabile significativi. Evita x,y se puoi scrivere base, altezza.

E4 — Pari o dispari?

Testo: Leggi un intero. Se è pari stampa "pari", altrimenti stampa "dispari".

Competenze: operatore resto %, if / else base.

// C++
#include <iostream>
int main() {
    int n;
    std::cout << "Numero intero: ";
    std::cin >> n;

    if (n % 2 == 0) {
        std::cout << "pari\n";
    } else {
        std::cout << "dispari\n";
    }
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int n;
    printf("Numero intero: ");
    scanf("%d", &n);

    if (n % 2 == 0) {
        printf("pari\n");
    } else {
        printf("dispari\n");
    }
    return 0;
}

n % 2 calcola il resto della divisione per 2. Se il resto è 0 → il numero è pari.

E5 — Minimo tra due numeri

Testo: Leggi due interi e stampa qual è il più piccolo. Se sono uguali, dillo chiaramente.

Competenze: confronto numerico con < e ==, if / else.

// C++
#include <iostream>
int main() {
    int a, b;
    std::cout << "Inserisci due numeri: ";
    std::cin >> a >> b;

    if (a < b) {
        std::cout << "Minore = " << a << "\n";
    } else if (b < a) {
        std::cout << "Minore = " << b << "\n";
    } else {
        std::cout << "Sono uguali\n";
    }
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int a, b;
    printf("Inserisci due numeri: ");
    scanf("%d %d", &a, &b);

    if (a < b) {
        printf("Minore = %d\n", a);
    } else if (b < a) {
        printf("Minore = %d\n", b);
    } else {
        printf("Sono uguali\n");
    }
    return 0;
}

Qui stai già ragionando in termini di casi esclusivi: a<b, b<a, oppure “nessuna delle due”, quindi uguali.

E6 — Conta da 1 a 10 (for)

Testo: Stampa i numeri da 1 a 10 (uno per riga) usando un ciclo for.

Competenze: ciclo for, incremento.

// C++
#include <iostream>
int main() {
    for (int i = 1; i <= 10; i++) {
        std::cout << i << "\n";
    }
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int i;
    for (i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }
    return 0;
}

Struttura tipica: for (inizio; condizione; aggiornamento). Qui: inizia da 1, finché ≤10, ogni giro fai i++.

E7 — Tabellina di un numero

Testo: Chiedi un intero n. Stampa la sua tabellina da n×1 fino a n×10.

Competenze: input, ciclo for, moltiplicazione.

// C++
#include <iostream>
int main() {
    int n;
    std::cout << "Numero: ";
    std::cin >> n;

    for (int i = 1; i <= 10; i++) {
        std::cout << n << " x " << i
                  << " = " << (n * i) << "\n";
    }
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int n, i;
    printf("Numero: ");
    scanf("%d", &n);

    for (i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", n, i, n * i);
    }
    return 0;
}

Questo è un pattern perfetto per iniziare a “generare tabelle” con un ciclo.

E8 — Numero positivo obbligatorio (while)

Testo: Chiedi all’utente un intero positivo (>0). Se inserisce un valore ≤0, richiedilo finché non è valido.

Competenze: ciclo while di validazione input.

// C++
#include <iostream>
int main() {
    int n;
    std::cout << "Inserisci un intero positivo: ";
    std::cin >> n;

    while (n <= 0) {
        std::cout << "Valore non valido. Riprova: ";
        std::cin >> n;
    }

    std::cout << "Hai inserito: " << n << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int n;
    printf("Inserisci un intero positivo: ");
    scanf("%d", &n);

    while (n <= 0) {
        printf("Valore non valido. Riprova: ");
        scanf("%d", &n);
    }

    printf("Hai inserito: %d\n", n);
    return 0;
}

Questo è un "ciclo di controllo input": finché l’utente sbaglia, non lo lasci uscire.

E9 — Media di 5 voti (array)

Testo: Leggi 5 voti interi in un array v. Calcola e stampa la media (anche decimale).

Competenze: array 1D, ciclo for, somma cumulativa, cast a double.

// C++
#include <iostream>
int main() {
    const int N = 5;
    int v[N];
    int somma = 0;

    for (int i = 0; i < N; i++) {
        std::cout << "Voto " << i << ": ";
        std::cin >> v[i];
        somma += v[i];
    }

    double media = (double)somma / N;
    std::cout << "Media = " << media << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 5;
    int v[N];
    int somma = 0;
    int i;

    for (i = 0; i < N; i++) {
        printf("Voto %d: ", i);
        scanf("%d", &v[i]);
        somma += v[i];
    }

    double media = (double)somma / N;
    printf("Media = %.2f\n", media);
    return 0;
}

Qui introduciamo l’idea “array = tanti valori con un solo nome”, e facciamo un’operazione globale sulla collezione.

E10 — Stampa l’array al contrario

Testo: Leggi 5 interi in un array a. Poi stampali in ordine inverso, dall’ultimo al primo.

Competenze: array, ciclo for in discesa, attenzione all’indice finale.

// C++
#include <iostream>
int main() {
    const int N = 5;
    int a[N];

    for (int i = 0; i < N; i++) {
        std::cout << "a[" << i << "] = ";
        std::cin >> a[i];
    }

    std::cout << "Ordine inverso: ";
    for (int i = N-1; i >= 0; i--) {
        std::cout << a[i] << " ";
    }
    std::cout << "\n";

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 5;
    int a[N];
    int i;

    for (i = 0; i < N; i++) {
        printf("a[%d] = ", i);
        scanf("%d", &a[i]);
    }

    printf("Ordine inverso: ");
    for (i = N-1; i >= 0; i--) {
        printf("%d ", a[i]);
    }
    printf("\n");
    return 0;
}

Occhio: il ciclo in discesa deve fermarsi dopo l’indice 0. Qui controlliamo i >= 0 e poi facciamo i--. Funziona perché i è int (cioè con segno). Evita per ora tipi unsigned nei contatori in discesa.

In sintesi
  • Se sai fare input/output, sai leggere i dati dal mondo.
  • Se sai usare if, puoi prendere decisioni.
  • Se sai usare for/while, puoi ripetere azioni e costruire tabelle di output.
  • Se sai usare gli array, puoi salvare e processare molti valori insieme.
