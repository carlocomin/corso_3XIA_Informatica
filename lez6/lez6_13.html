Esercizi intermedi — Decisioni, array, cicli annidati

Obiettivo della lezione

Qui iniziamo a combinare più concetti insieme:

  • if / else con intervalli di valori;
  • switch + do-while per menù ripetuti;
  • tecniche di conteggio e ricerca in un array;
  • massimo/minimo in una collezione di dati;
  • cicli annidati (for dentro for) per creare tabelle bidimensionali;
  • prime matrici 3×3: inserire, scorrere, sommare.

Questi esercizi sono molto vicini al “codice reale”: iniziano a sembrare piccoli strumenti utili, non più solo mini-esempi.

E1 — Classificazione del voto

Testo: Leggi un voto intero tra 0 e 10 e stampa:

  • <6 → "insufficiente"
  • 6–7 → "sufficiente"
  • 8–9 → "buono"
  • 10 → "ottimo"
Se è fuori dall'intervallo 0–10, segnala errore.

Competenze: catena di if / else if / else e gestione intervalli numerici.

// C++
#include <iostream>
int main() {
    int voto;
    std::cout << "Inserisci voto (0-10): ";
    std::cin >> voto;

    if (voto < 0 || voto > 10) {
        std::cout << "Valore non valido\n";
    } else if (voto < 6) {
        std::cout << "insufficiente\n";
    } else if (voto <= 7) {
        std::cout << "sufficiente\n";
    } else if (voto <= 9) {
        std::cout << "buono\n";
    } else {
        std::cout << "ottimo\n";
    }
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int voto;
    printf("Inserisci voto (0-10): ");
    scanf("%d", &voto);

    if (voto < 0 || voto > 10) {
        printf("Valore non valido\n");
    } else if (voto < 6) {
        printf("insufficiente\n");
    } else if (voto <= 7) {
        printf("sufficiente\n");
    } else if (voto <= 9) {
        printf("buono\n");
    } else {
        printf("ottimo\n");
    }
    return 0;
}

Nota l’uso di || (“oppure” logico) per controllare subito i casi fuori range.

E2 — Menù interattivo con switch e do-while

Testo: Mostra un menù con 3 voci:

  1. Saluta
  2. Mostra numero segreto
  3. Esci
Dopo ogni scelta (tranne Esci) il menù deve ricomparire finché l’utente non sceglie 3.

Competenze: switch, ciclo do-while, gestione input utente ripetuto.

// C++
#include <iostream>
int main() {
    int scelta;
    do {
        std::cout << "\nMenu:\n"
                  << "1) Saluta\n"
                  << "2) Numero segreto\n"
                  << "3) Esci\n"
                  << "Scelta: ";
        std::cin >> scelta;

        switch (scelta) {
            case 1:
                std::cout << "Ciao!\n";
                break;
            case 2:
                std::cout << "Il numero segreto e' 42\n";
                break;
            case 3:
                std::cout << "Uscita...\n";
                break;
            default:
                std::cout << "Scelta non valida\n";
                break;
        }
    } while (scelta != 3);

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int scelta;
    do {
        printf("\nMenu:\n");
        printf("1) Saluta\n");
        printf("2) Numero segreto\n");
        printf("3) Esci\n");
        printf("Scelta: ");
        scanf("%d", &scelta);

        switch (scelta) {
            case 1:
                printf("Ciao!\n");
                break;
            case 2:
                printf("Il numero segreto e' 42\n");
                break;
            case 3:
                printf("Uscita...\n");
                break;
            default:
                printf("Scelta non valida\n");
                break;
        }
    } while (scelta != 3);

    return 0;
}

Questo schema è molto importante: è la base di tutti i piccoli programmi “a menù”.

E3 — Conta le cifre di un numero

Testo: Leggi un intero positivo e stampa quante cifre ha (es: 7 → 1 cifra; 123 → 3 cifre).

Competenze: ciclo while che divide per 10 finché il numero non arriva a 0; gestione caso speciale 0.

// C++
#include <iostream>
int main() {
    int x;
    std::cout << "Numero positivo: ";
    std::cin >> x;

    if (x == 0) {
        std::cout << "Cifre: 1\n";
        return 0;
    }

    int count = 0;
    while (x != 0) {
        x = x / 10; // elimina l'ultima cifra
        count++;
    }

    std::cout << "Cifre: " << count << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int x;
    printf("Numero positivo: ");
    scanf("%d", &x);

    if (x == 0) {
        printf("Cifre: 1\n");
        return 0;
    }

    int count = 0;
    while (x != 0) {
        x = x / 10;
        count++;
    }

    printf("Cifre: %d\n", count);
    return 0;
}

Questo è un esempio di “ciclo che termina quando la variabile diventa 0”. Molto usato nell’analisi di numeri.

E4 — Massimo e minimo in un array

Testo: Leggi 5 numeri interi in un array. Stampa il massimo e il minimo trovati.

Competenze: array + ciclo for + confronto progressivo (salvo il primo valore e poi confronto).

// C++
#include <iostream>
int main() {
    const int N = 5;
    int v[N];

    for (int i = 0; i < N; i++) {
        std::cout << "v[" << i << "] = ";
        std::cin >> v[i];
    }

    int minimo = v[0];
    int massimo = v[0];

    for (int i = 1; i < N; i++) {
        if (v[i] < minimo) minimo = v[i];
        if (v[i] > massimo) massimo = v[i];
    }

    std::cout << "Min = " << minimo
              << ", Max = " << massimo << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 5;
    int v[N];
    int i;

    for (i = 0; i < N; i++) {
        printf("v[%d] = ", i);
        scanf("%d", &v[i]);
    }

    int minimo = v[0];
    int massimo = v[0];

    for (i = 1; i < N; i++) {
        if (v[i] < minimo) minimo = v[i];
        if (v[i] > massimo) massimo = v[i];
    }

    printf("Min = %d, Max = %d\n", minimo, massimo);
    return 0;
}

Pattern molto comune: 1) prendi il primo valore come riferimento, 2) scorri gli altri confrontando.

E5 — Ricerca lineare in un array

Testo: Leggi 5 numeri in un array. Poi chiedi un numero X e stampa se X è presente e a quale indice (prima occorrenza).

Competenze: ricerca lineare con for, uso di una variabile-flag e di break.

// C++
#include <iostream>
int main() {
    const int N = 5;
    int v[N];
    for (int i = 0; i < N; i++) {
        std::cout << "v[" << i << "] = ";
        std::cin >> v[i];
    }

    int x;
    std::cout << "Cerca: ";
    std::cin >> x;

    bool trovato = false;
    int posizione = -1;

    for (int i = 0; i < N; i++) {
        if (v[i] == x) {
            trovato = true;
            posizione = i;
            break; // fermo la ricerca
        }
    }

    if (trovato) {
        std::cout << "Trovato a indice " << posizione << "\n";
    } else {
        std::cout << "Non trovato\n";
    }

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 5;
    int v[N];
    int i;

    for (i = 0; i < N; i++) {
        printf("v[%d] = ", i);
        scanf("%d", &v[i]);
    }

    int x;
    printf("Cerca: ");
    scanf("%d", &x);

    int trovato = 0;
    int posizione = -1;

    for (i = 0; i < N; i++) {
        if (v[i] == x) {
            trovato = 1;
            posizione = i;
            break;
        }
    }

    if (trovato) {
        printf("Trovato a indice %d\n", posizione);
    } else {
        printf("Non trovato\n");
    }

    return 0;
}

Questo è il classico algoritmo di ricerca lineare. Complessità concettuale: controlla ogni elemento, uno alla volta.

E6 — Somma elemento per elemento di due array

Testo: Leggi due array A e B di 5 interi ciascuno, poi costruisci un terzo array C tale che C[i] = A[i] + B[i]. Infine stampa C.

Competenze: uso parallelo di più array e indicizzazione coerente.

// C++
#include <iostream>
int main() {
    const int N = 5;
    int A[N], B[N], C[N];

    std::cout << "Inserisci A:\n";
    for (int i = 0; i < N; i++) {
        std::cout << "A[" << i << "] = ";
        std::cin >> A[i];
    }

    std::cout << "Inserisci B:\n";
    for (int i = 0; i < N; i++) {
        std::cout << "B[" << i << "] = ";
        std::cin >> B[i];
    }

    for (int i = 0; i < N; i++) {
        C[i] = A[i] + B[i];
    }

    std::cout << "C = ";
    for (int i = 0; i < N; i++) {
        std::cout << C[i] << " ";
    }
    std::cout << "\n";

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 5;
    int A[N], B[N], C[N];
    int i;

    printf("Inserisci A:\n");
    for (i = 0; i < N; i++) {
        printf("A[%d] = ", i);
        scanf("%d", &A[i]);
    }

    printf("Inserisci B:\n");
    for (i = 0; i < N; i++) {
        printf("B[%d] = ", i);
        scanf("%d", &B[i]);
    }

    for (i = 0; i < N; i++) {
        C[i] = A[i] + B[i];
    }

    printf("C = ");
    for (i = 0; i < N; i++) {
        printf("%d ", C[i]);
    }
    printf("\n");

    return 0;
}

Questo è il primo passo verso l’elaborazione di vettori/matrici numeriche.

E7 — Tabella pitagorica 1..10 (cicli annidati)

Testo: Stampa una tabella 10×10 in cui la cella (r,c) contiene r * c. Le righe e le colonne vanno da 1 a 10.

Competenze: cicli for annidati, formattazione tabellare, stampa su righe multiple.

// C++
#include <iostream>
int main() {
    for (int r = 1; r <= 10; r++) {
        for (int c = 1; c <= 10; c++) {
            std::cout << (r * c) << "\t";
        }
        std::cout << "\n";
    }
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int r, c;
    for (r = 1; r <= 10; r++) {
        for (c = 1; c <= 10; c++) {
            printf("%d\t", r * c);
        }
        printf("\n");
    }
    return 0;
}

Questo è un esempio “classico” di due for annidati: uno per le righe, uno per le colonne.

E8 — Matrice 3×3: somma di tutti gli elementi

Testo: Chiedi 9 numeri interi all’utente e salvali in una matrice 3×3. Poi calcola la somma totale di tutti gli elementi e stampala.

Competenze: array bidimensionale, doppio for, accumulatore di somma.

// C++
#include <iostream>
int main() {
    int mat[3][3];
    int somma = 0;

    // input
    for (int r = 0; r < 3; r++) {
        for (int c = 0; c < 3; c++) {
            std::cout << "mat[" << r << "][" << c << "] = ";
            std::cin >> mat[r][c];
        }
    }

    // somma
    for (int r = 0; r < 3; r++) {
        for (int c = 0; c < 3; c++) {
            somma += mat[r][c];
        }
    }

    std::cout << "Somma totale = " << somma << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int mat[3][3];
    int somma = 0;
    int r, c;

    // input
    for (r = 0; r < 3; r++) {
        for (c = 0; c < 3; c++) {
            printf("mat[%d][%d] = ", r, c);
            scanf("%d", &mat[r][c]);
        }
    }

    // somma
    for (r = 0; r < 3; r++) {
        for (c = 0; c < 3; c++) {
            somma += mat[r][c];
        }
    }

    printf("Somma totale = %d\n", somma);
    return 0;
}

Una matrice 3×3 è solo un array di 3 righe, ognuna contenente 3 elementi. Si accede con due indici: riga e colonna.

E9 — Diagonale principale (3×3)

Testo: Dopo aver letto una matrice 3×3 di interi, stampa gli elementi sulla diagonale principale: quelli con riga == colonna, cioè mat[0][0], mat[1][1], mat[2][2].

Competenze: comprensione del concetto di diagonale; ciclo su i unico.

// C++
#include <iostream>
int main() {
    int mat[3][3];

    for (int r = 0; r < 3; r++) {
        for (int c = 0; c < 3; c++) {
            std::cout << "mat[" << r << "][" << c << "] = ";
            std::cin >> mat[r][c];
        }
    }

    std::cout << "Diagonale principale: ";
    for (int i = 0; i < 3; i++) {
        std::cout << mat[i][i] << " ";
    }
    std::cout << "\n";

    return 0;
}
// C
#include <stdio.h>
int main(void) {
    int mat[3][3];
    int r, c;

    for (r = 0; r < 3; r++) {
        for (c = 0; c < 3; c++) {
            printf("mat[%d][%d] = ", r, c);
            scanf("%d", &mat[r][c]);
        }
    }

    printf("Diagonale principale: ");
    for (r = 0; r < 3; r++) {
        printf("%d ", mat[r][r]);
    }
    printf("\n");

    return 0;
}

Concetto fondamentale: in molti algoritmi su matrici, “i == j” significa essere sulla diagonale principale.

E10 — Conta quanti elementi superano una soglia

Testo: Leggi 10 interi in un array. Poi leggi un intero T (soglia). Stampa quanti elementi dell’array sono > T.

Competenze: scansione dell’array e conteggio con un accumulatore.

// C++
#include <iostream>
int main() {
    const int N = 10;
    int v[N];
    int T;

    for (int i = 0; i < N; i++) {
        std::cout << "v[" << i << "] = ";
        std::cin >> v[i];
    }

    std::cout << "Soglia T: ";
    std::cin >> T;

    int count = 0;
    for (int i = 0; i < N; i++) {
        if (v[i] > T) {
            count++;
        }
    }

    std::cout << "Elementi > T: " << count << "\n";
    return 0;
}
// C
#include <stdio.h>
int main(void) {
    const int N = 10;
    int v[N];
    int i, T;

    for (i = 0; i < N; i++) {
        printf("v[%d] = ", i);
        scanf("%d", &v[i]);
    }

    printf("Soglia T: ");
    scanf("%d", &T);

    int count = 0;
    for (i = 0; i < N; i++) {
        if (v[i] > T) {
            count++;
        }
    }

    printf("Elementi > T: %d\n", count);
    return 0;
}

Questo esercizio allena l’idea di “scandire l’array e contare quanti soddisfano una certa condizione”. È una tecnica usatissima in analisi dati.
